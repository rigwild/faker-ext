# Faker Extension

A browser extension to host your social network content externally.

LinkedIn will be focused first as a proof of concept.

## Supported Social Networks

- LinkedIn
  - [x] Post
  - [ ] Image
- Facebook
  - [ ] Post
  - [ ] Image
- Twitter
  - [ ] Post
  - [ ] Image
- Instagram
  - [ ] Image
- Reddit
  - [ ] Post
  - [ ] Image
