Welcome to the Faker server!

Get a Post

Request

GET /faker/api/posts/:id

Response

{
  "id": "12345678-abcd-4abc-abcd-123456789abc",
  "content": "Hello world!", 
  "createdAt": "2022-02-17T17:52:07.463Z",
  "updatedAt": "2022-02-17T17:52:07.463Z"
}

Get a Media

Request

GET /faker/api/medias/:id

Response

<file data>

Add a Post

Request

POST /faker/api/upload
Authorization: Basic <base64 of username:password>
Content-Type: application/json

{ "content": "hello" }

Response

{
  "externalUri": "/faker/api/posts/12345678-abcd-4abc-abcd-123456789abc"
}

Add a Media

Request

The FormData key containing your file should be media.

Supported media MIME types:
  • image/jpeg
  • image/png
  • image/webp
  • video/mp4
POST /faker/api/upload
Authorization: Basic <base64 of username:password>
Content-Type: multipart/form-data

Response

{
  "externalUri": "/faker/api/medias/12345678-abcd-4abc-abcd-123456789abc"
}
